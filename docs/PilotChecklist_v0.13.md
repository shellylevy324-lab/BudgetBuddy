# v0.13 Pilot Checklist

## Teacher Profile
- Select Prompt Fading + Token Goal
- Choose a starting prompt level
- Set token goal and maximum trials
- Optionally enter local approved media paths
- Save, refresh, and confirm settings persist

## Student Session
- Token board appears
- Buttons remain disabled until the programmed prompt finishes
- Correct first response earns exactly one token
- Incorrect first response earns no token
- Model prompt highlights the correct answer
- Student completes the modeled response before the next trial
- Session ends at token goal or maximum trials

## Media
- Blank media paths use the built-in star animation
- Optional GIF/image displays briefly after token earning
- Optional completion audio plays only after browser interaction
- Reduced-motion system preference removes built-in animation

## Baseline Regression
- No token board
- No prompts, correction, or reinforcement
