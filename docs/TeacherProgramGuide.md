# Student Program Guide — v0.14

## Purpose
The Student Programs area organizes Budget Buddy settings in the same functional categories commonly used in ABA teaching protocols.

## Sections
- **Program Information:** student name and session size
- **Teaching Procedure:** Baseline, Least-to-Most, or Prompt Fading + Token Goal
- **Skill Settings:** Budget Buddy level, item counts, and budget difficulty
- **SD, Prompting & Timing:** SD preview, response delay, independent wait, prompt spacing, and starting prompt
- **Reinforcement:** praise, chime, tokens, preferred media, and celebration duration
- **Error Correction:** current correction sequence, displayed for staff reference
- **Data Collection:** measures recorded automatically by the application

## Important
The v0.14 redesign changes organization and language only. It does not automatically make instructional decisions or fade prompts. The teacher remains responsible for selecting and changing the teaching procedure.
