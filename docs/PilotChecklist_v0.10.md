# v0.10 Pilot Checklist

## Student Profile
- Select Random or Custom Shopping List
- For Random, select 1–5 items
- For Custom, enter one exact grocery item name per line
- Save the student profile
- Refresh and confirm settings remain

## Session
- Shopping list appears on the student screen
- Current item is highlighted
- Item image and name match the current list item
- Completed items receive a checkmark
- Items-remaining count updates
- Session ends after the final list item

## Baseline
- No trial-by-trial reinforcement
- No prompts
- Shopping-list progress still updates
- Completion animation appears

## Teaching
- Prompt sequence still works
- Reinforcement still follows the student profile
- Shopping-list progress updates after each answer

## Teacher Platform
- Session data saves
- Export/import includes shopping-list settings
