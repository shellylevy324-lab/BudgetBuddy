# v0.8 Pilot Checklist

## Before a Student Session
- Confirm the correct student profile is selected
- Confirm Baseline or Least-to-Most mode
- Confirm trial count
- Confirm audio setting
- Confirm reinforcement profile
- Confirm device volume
- Run one test trial

## Baseline Test
- No prompts appear
- No praise or chime follows correct responses
- No error correction follows incorrect responses
- Trials advance automatically
- Completion animation appears at the end

## Teaching Test
- Prompt sequence appears at the correct times
- Correct reinforcement matches the student profile
- Incorrect responses receive neutral feedback
- Trials advance automatically
- Completion animation appears at the end

## Teacher Platform Test
- Session report saves
- Latency appears
- Prompt level appears
- Independent percentage appears
- Export works
