# v0.16 Pilot Checklist

## Create a Package
- Open Classroom → Reinforcement Library
- Add a package name and praise
- Upload token media
- Upload completion media and audio
- Preview both reinforcers
- Save the package

## Assign to Students
- Open two student programs
- Assign the same classroom package to both
- Save and refresh
- Confirm both students retain the package
- Run sessions and confirm the shared media appears

## Edit and Deactivate
- Edit the shared package
- Confirm assigned students use the updated package
- Deactivate the package
- Confirm it disappears from new choices
- Confirm saved data remains intact

## Backup and Restore
- Export a classroom backup
- Restore it in a test browser
- Confirm classroom, staff, library, students, and sessions return

## Regression
- Student Programs opens
- Start Shopping works
- Administrator selection works
- Built-in packages preview
- Student-specific custom media still works
- Levels 1–3 launch
