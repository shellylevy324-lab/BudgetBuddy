# Portable Classroom Guide — v0.17

## Teacher device
1. Make and save classroom, staff, reinforcement, or student-program changes.
2. Open **Import / Export**.
3. Confirm the classroom revision increased.
4. Select **Export Portable Classroom**.
5. Move the `.bbclass` file to the student device using an approved school method.

## Student device
1. Open the same deployed Budget Buddy website.
2. Open **Student Programs → Import / Export**.
3. Keep **Update teaching setup and keep this device's session reports** selected.
4. Choose the newest `.bbclass` file and confirm the revision.
5. Start the student's session.

## Returning data to the teacher device
Export the portable classroom from the student device and import it on the teacher device using **Replace the entire classroom, including reports** only when you intentionally want the reports in that file to replace the teacher device's reports.

This release is manual transfer, not live cloud synchronization.
