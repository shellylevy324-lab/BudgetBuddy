# v0.12 Pilot Checklist

## Teacher Profile
- Select Level 3 — Choose Items Within Budget
- Choose 2–4 required items
- Choose 4–5 available choices
- Save and refresh to confirm settings persist

## Student Session
- Choice cards show image, item name, and price
- Tapping selects and deselects an item
- Selection count updates
- Cart Total updates
- Money Left updates
- Check My Cart stays disabled until the required number is selected

## Teaching Mode
- An within-budget cart advances the trial
- An over-budget cart provides neutral correction
- Student can revise the same cart
- Reinforcement follows the student's profile

## Baseline
- First submitted cart is recorded
- No correction or reinforcement appears
- Session advances immediately

## Reports
- Selected item names are saved
- Cart total and budget are saved
- Latency and prompt level are saved
- Cart attempt count is included in the stored trial record
